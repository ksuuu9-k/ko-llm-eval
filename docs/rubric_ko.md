# Korean Rubric Draft

초기 MVP 단계에서는 아래 항목을 중심으로 rubric을 구체화한다.

- tone
- fluency
- grounding

## Tone

- 높음: 존댓말 또는 반말이 일관되고, 서비스 맥락에 맞는 공손함을 유지함
- 보통: 전반적으로 일관되지만 일부 표현이 어색하거나 높임 수준이 흔들림
- 낮음: 존댓말/반말 혼용, 문장 종결 어미 불일치, 서비스 톤에 부적합

## Fluency

- 높음: 한국어 원어민 기준 자연스럽고 매끄러움
- 보통: 의미는 전달되지만 번역투나 딱딱한 표현이 일부 존재
- 낮음: 부자연스럽거나 파편적이며 비원어민식 표현이 두드러짐

## Grounding

- 높음: 답변이 주어진 context에 기반하고 추가 주장에 근거가 있음
- 보통: 대부분 context 기반이지만 일부 과한 일반화가 있음
- 낮음: context에 없는 내용이 추가되거나 사실이 왜곡됨
